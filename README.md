Le Crabe Migrateur - Site de Randonnée

Projet de développement web Full Stack conçu pour guider les passionnés de nature à travers les sentiers sauvages. Ce site propose des itinéraires détaillés, des cartes interactives et une interface communautaire.

installation

    1.Cloner le projet :

    url: https://github.com/Ahayahamane/Le_Crabe_Migrateur.git

    2.Installer les dépendances :

    composer install

    3.Importer la base de données
    
    1. Modifiez le fichier .env avec vos identifiants de connexion à la base de données :

    DB_HOST=votre-serveur
    DB_NAME=crabe_migrateur
    DB_USER=votre_utilisateur
    DB_PASS=votre_mot_de_passe
